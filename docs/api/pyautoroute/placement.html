pyautoroute.placement

Experimental simulated-annealing footprint placement.

An opt-in pass (--place) that arranges the board's footprints before routing, the placement analogue of pyautoroute.anneal: where the annealer moves tracks, this moves footprint positions/rotations to minimise rats-nest length while keeping bodies from overlapping and pulling the layout together.

Energy ``E = ratsnest + overlap_weight·overlap_area + compact_weight·bbox_area

  • edge_weight·edge_distance + containment_weight·area_outside_outline
  • congestion_weight·Σ field(centroid)``:
  • ratsnest — total MST length over the pad centroids (reuses pyautoroute.netlist); shrinks as connected pads are drawn together.
  • overlap_area — pairwise intersection of footprint body boxes, found via a shapely STRtree. Each box is inflated by half of buffer per side, so a pair registers as overlapping until its gap exceeds buffer; the optimiser then keeps footprints at least buffer apart, leaving room for routing clearance (this is the fix for placements packing so tightly that the routed board failed DRC). A pair where either footprint opted in via the Autoroute-overlap property (pcb.Footprint.overlap_ok) contributes only its pad-vs-pad overlap (also buffer-inflated), not body overlap — the shield-over-board case. Each footprint box also covers its visible silkscreen text, and standalone board-level silk text (gr_text pin labels, title block) is added as fixed keep-out boxes, so footprints aren't placed on top of existing silkscreen.
  • bbox_area — area of the bounding box of all footprints; compaction emerges from this term under cooling, with no separate phase.
  • edge_distance — only for footprints that opt in via the Autoroute-edge property (any for the nearest side, or left / right / top / bottom for a named one; pcb.Footprint.edge_affinity). Each flagged footprint is penalised by the distance from its target side of the current layout bounding box to the far side of its box — i.e. the gap to the edge plus the box's depth perpendicular to it — so the term both pulls connectors and the like onto the board boundary and orients them to lie flat against it (long axis parallel) rather than rotating to reach the edge with a single pad. Measured against the layout bbox (not absolute coordinates), so it stays translation-invariant like the other terms. Zero when nothing is flagged, so the default behaviour is unchanged.
  • area_outside_outline — only under keep_outline (the --keep-outline mode), and only when the board has a closed Edge.Cuts. Penalises the area each footprint box protrudes outside that fixed outline, containing the placement within the existing board shape rather than regenerating a bounding box; edge affinity then targets the outline rather than the layout bbox. Zero otherwise. Note keep_outline makes the energy depend on absolute position (the outline is fixed), so recenter is skipped in that mode.
  • Σ field(centroid) — only under congestion feedback (--place-feedback), when a pyautoroute.router.CongestionField from a previous cycle's routed result is supplied. Each footprint's body-box centre is sampled in the field (high where routing was congested — dense copper, vias, unrouted nets) and the values summed, so minimising the term spreads parts out of the hot cells the router struggled with. The field is in absolute board coordinates (it anchors the layout), so like keep_outline it makes the energy position-dependent and recenter is skipped. Zero when no field is supplied, so default behaviour is unchanged.

Moves (over the movable footprints — locked ones are fixed obstacles): translate by a temperature-scaled random step, rotate (rotate_mode: ortho = ±90°/180°, free = any angle, none = no rotation), or swap two footprints' origins. Worse moves are accepted with Metropolis probability under a geometric t_start → t_end schedule; the best-seen placement is kept and left on the board. Pad absolute coordinates are kept in sync on every move (pcb.Footprint.sync_pads) so the energy geometry stays consistent; after the run the caller applies pcb.apply_placement to finalise the pads and board outline before routing.

The energy depends only on the footprints' relative poses, so it is translation-invariant and an unlocked cluster random-walks (drifts) during the run. place therefore calls recenter before returning, shifting the placement rigidly back onto its starting centroid without changing any energy term.

 93def _fp_silk_text_extents(fp: Footprint) -> list[tuple[float, float, float]]:
 94    """Return ``(local_x, local_y, half_diag)`` for each visible silkscreen text.
 95
 96    The ``(local_x, local_y)`` are in the footprint's local coordinate frame
 97    (pre-rotation); ``half_diag`` is a rotation-invariant radius estimate based
 98    on the text content length and declared font height, so the extent can be
 99    applied without knowing the text's own angle.
100
101    Args:
102        fp: the footprint whose text nodes are scanned.
103
104    Returns:
105        One tuple per visible, non-hidden silkscreen text item (Reference or
106        Value property, or fp_text) in the footprint.
107    """
108    from .pcb import children, child, strings, floats, atoms_after_head
109    from .sexpr import SList
110
111    extents: list[tuple[float, float, float]] = []
112
113    def _hidden(node) -> bool:
114        h = child(node, "hide")
115        if h is None:
116            return False
117        vals = atoms_after_head(h)
118        return bool(vals) and vals[0].text == "yes"
119
120    def _layer(node) -> str:
121        ls = strings(child(node, "layer"))
122        return ls[0] if ls else ""
123
124    def _font_h(node) -> float:
125        eff = child(node, "effects")
126        if eff is None:
127            return 1.0
128        fnt = child(eff, "font")
129        if fnt is None:
130            return 1.0
131        sz = child(fnt, "size")
132        vals = floats(sz) if sz is not None else []
133        return vals[0] if vals else 1.0
134
135    def _add(content: str, at_vals, fh: float) -> None:
136        if len(at_vals) < 2 or not content:
137            return
138        w = len(content) * fh * 0.7
139        h = fh * 1.3
140        extents.append((at_vals[0], at_vals[1], 0.5 * math.hypot(w, h)))
141
142    fp_node = fp.fp_node
143
144    for prop in children(fp_node, "property"):
145        if _hidden(prop) or _layer(prop) not in _SILK_LAYERS:
146            continue
147        atoms = atoms_after_head(prop)
148        if len(atoms) < 2 or atoms[0].text not in ("Reference", "Value"):
149            continue
150        content = atoms[1].text
151        _add(content, floats(child(prop, "at")), _font_h(prop))
152
153    for txt in children(fp_node, "fp_text"):
154        if _hidden(txt) or _layer(txt) not in _SILK_LAYERS:
155            continue
156        atoms = atoms_after_head(txt)
157        if len(atoms) < 2:
158            continue
159        content = atoms[1].text.replace("${REFERENCE}", fp.ref)
160        _add(content, floats(child(txt, "at")), _font_h(txt))
161
162    return extents

165def _board_silk_text_boxes(board: Board) -> list[tuple[float, float, float]]:
166    """Return ``(cx, cy, half_diag)`` for each visible board-level silk text.
167
168    Board-level ``gr_text`` items — connector pin labels, a title block, etc. —
169    are not footprints, so the placer would otherwise ignore them and happily
170    drop a footprint on top (the locked "Bus Indicator" / pin-label text on the
171    Test1 board). Each tuple is the text's centre and a rotation-invariant
172    half-diagonal radius of its extent, in board coordinates, mirroring the
173    estimate `_fp_silk_text_extents` uses for footprint text, so footprints can
174    be pushed clear of the text.
175
176    Args:
177        board: the board whose top-level ``gr_text`` nodes are scanned.
178
179    Returns:
180        One tuple per visible, non-hidden silkscreen ``gr_text`` on the board.
181    """
182    from .pcb import children, child, strings, floats, atoms_after_head
183
184    out: list[tuple[float, float, float]] = []
185    for txt in children(board.tree, "gr_text"):
186        h = child(txt, "hide")
187        if h is not None:
188            vals = atoms_after_head(h)
189            if vals and vals[0].text == "yes":
190                continue
191        ls = strings(child(txt, "layer"))
192        if not ls or ls[0] not in _SILK_LAYERS:
193            continue
194        atoms = atoms_after_head(txt)
195        if not atoms:
196            continue
197        content = atoms[0].text
198        at = floats(child(txt, "at"))
199        if not content or len(at) < 2:
200            continue
201        x, y, angle = at[0], at[1], (at[2] if len(at) >= 3 else 0.0)
202
203        fh = 1.0
204        eff = child(txt, "effects")
205        fnt = child(eff, "font") if eff is not None else None
206        sz = child(fnt, "size") if fnt is not None else None
207        szv = floats(sz) if sz is not None else []
208        if szv:
209            fh = szv[0]
210
211        # Estimate the unrotated text extent, then offset the box centre from
212        # the `at` anchor per the justify (KiCad's `at` is an edge/corner when
213        # justified, the centre otherwise); y grows downward.
214        w = len(content) * fh * 0.7
215        th = fh * 1.3
216        just_node = child(eff, "justify") if eff is not None else None
217        just = {a.text for a in atoms_after_head(just_node)} if just_node else set()
218        dx = 0.5 * w if "left" in just else (-0.5 * w if "right" in just else 0.0)
219        dy = -0.5 * th if "bottom" in just else (0.5 * th if "top" in just else 0.0)
220        cos_a = math.cos(math.radians(angle))
221        sin_a = math.sin(math.radians(angle))
222        cx = x + dx * cos_a + dy * sin_a
223        cy = y - dx * sin_a + dy * cos_a
224        out.append((cx, cy, 0.5 * math.hypot(w, th)))
225    return out

228@dataclass
229class PlaceParams:
230    iters: int | None = None
231    time_budget: float | None = None
232    t_start: float = 8.0
233    t_end: float = 0.05
234    overlap_weight: float = 20.0      # mm-equivalent cost per mm² of body/pad overlap
235    compact_weight: float = 0.02      # mm-equivalent cost per mm² of layout bbox
236    edge_weight: float = 2.0          # mm-cost per mm an edge-flagged footprint sits from its target edge
237    keep_outline: bool = False        # contain footprints within the board's existing Edge.Cuts
238    containment_weight: float = 50.0  # cost per mm² a footprint protrudes outside the kept outline
239    # Congestion feedback (--place-feedback): when `congestion_field` is set (a
240    # `router.CongestionField` from a previous cycle's routed result), each
241    # footprint is penalised by `congestion_weight · field(centroid)`, pushing
242    # parts out of the cells where routing struggled. Absolute-coordinate, so the
243    # field anchors the layout — `recenter` is skipped while it is active. None /
244    # weight 0 (the default) leaves placement unchanged.
245    congestion_field: object = None
246    congestion_weight: float = 0.0    # mm-cost per unit field value at a footprint centroid
247    step: float = 20.0                # max translate step (mm) at t_start
248    buffer: float = 0.5               # keep-out gap (mm) enforced between footprints
249    rotate_mode: str = "ortho"        # "ortho" (+/-90/180), "free" (any angle), "none"
250    seed: int = 0
251    exclude: list[str] = field(default_factory=list)
252    # Early-termination (stall detection): if the windowed accept ratio stays
253    # below `stall_ratio` for `stall_patience` consecutive full accept-windows,
254    # the run stops early. Disabled when `stall_patience <= 0` or
255    # `stall_ratio <= 0` (the default), so the full budget is honoured.
256    stall_ratio: float = 0.02
257    stall_patience: int = 0

278def _half_extent(pad: Pad) -> float:
279    """Rotation-independent half-extent of a pad (half its bounding diagonal)."""
280    return 0.5 * math.hypot(pad.w, pad.h)

283class _Placer:
284    def __init__(self, board: Board, params: PlaceParams):
285        """Set up the placer over a board's footprints.
286
287        Args:
288            board: the board whose footprint poses are optimised in place.
289            params: the placement parameters (budget, schedule, weights, step).
290        """
291        self.board = board
292        self.p = params
293        self.rng = random.Random(params.seed)
294        self.boxed = [fp for fp in board.footprints if fp.pads]
295        self.movable = [fp for fp in self.boxed if not fp.locked]
296        # Each body/pad box is grown by half the buffer per side, so two boxes
297        # register as overlapping whenever their *gap* is below the full buffer —
298        # the optimiser then pushes footprints at least `buffer` apart, leaving
299        # room for the routing clearance and avoiding the too-tight placements
300        # that previously failed DRC.
301        self.half_buffer = max(0.0, params.buffer) / 2.0
302        # Precompute silkscreen text local extents once; looked up by id(fp).
303        self._text_extents: dict[int, list[tuple[float, float, float]]] = {
304            id(fp): _fp_silk_text_extents(fp) for fp in self.boxed
305        }
306        # Fixed board-level silkscreen text (pin labels, title block): static
307        # keep-out boxes footprints must avoid, so they aren't dropped on top.
308        self._fixed_text = [
309            box(cx - hr - self.half_buffer, cy - hr - self.half_buffer,
310                cx + hr + self.half_buffer, cy + hr + self.half_buffer)
311            for cx, cy, hr in _board_silk_text_boxes(board)
312        ]
313        self._fixed_tree = STRtree(self._fixed_text) if self._fixed_text else None
314
315        # Incremental-energy cache (populated by `_rebuild_cache`).
316        #
317        # The ratsnest topology is fixed for the whole run: a move changes pad
318        # *positions*, never which pads are connected, so the connection list and
319        # the footprint->incident-connection index are built once, and a move
320        # recomputes only the lengths of the connections touching the moved
321        # footprints. Overlap is cached per box and updated for the moved boxes
322        # against their neighbours; the bbox is recomputed from the cached box
323        # bounds (O(N), cheap relative to the old O(P^2) MST rebuild).
324        self._boxes: list = []            # per-footprint body box, parallel to self.boxed
325        self._bounds: list = []           # per-box (minx,miny,maxx,maxy), parallel to _boxes
326        self._conns: list = []            # cached netlist connections (fixed topology)
327        self._conn_len: list = []         # cached per-connection length, parallel to _conns
328        self._fp_conns: dict[int, list[int]] = {}   # boxed-index -> incident conn indices
329        self._rats = 0.0
330        self._overlap = 0.0
331        self._bbox = 0.0
332        self._edge = 0.0
333        self._containment = 0.0
334        self._congestion = 0.0
335        self._idx_of_fp: dict[int, int] = {id(fp): i for i, fp in enumerate(self.boxed)}
336        # Footprints that opted into edge placement: boxed-index -> side
337        # ("any"|"left"|"right"|"top"|"bottom"). Fixed for the run.
338        self._flagged: dict[int, str] = {
339            i: fp.edge_affinity for i, fp in enumerate(self.boxed) if fp.edge_affinity
340        }
341        # --keep-outline: contain footprints within the board's existing Edge.Cuts
342        # (rather than regenerating a bounding box). The polygon and its bounds are
343        # fixed for the run; edge affinity then targets the outline, not the layout
344        # bbox. Falls back to no containment if the board has no closed outline.
345        self._outline_poly = None
346        self._outline_bounds = None
347        if params.keep_outline and board.outline and not board.outline_synthesized:
348            try:
349                from . import geometry
350                self._outline_poly = geometry.outline_to_polygon(board.outline)
351                self._outline_bounds = self._outline_poly.bounds
352            except Exception:                 # no closed outline -> no containment
353                self._outline_poly = None
354                self._outline_bounds = None
355
356    def _build_index(self) -> None:
357        """Build the fixed connection list and footprint->connection incidence.
358
359        Called once: the ratsnest topology does not change across moves, so the
360        connection list (and which footprint each connection's pads belong to)
361        is computed a single time.
362        """
363        self._conns = netlist.build_connections(self.board, self.p.exclude)
364        pad_to_fp: dict[int, int] = {}
365        for i, fp in enumerate(self.boxed):
366            for pad in fp.pads:
367                pad_to_fp[id(pad)] = i
368        self._fp_conns = {i: [] for i in range(len(self.boxed))}
369        for ci, c in enumerate(self._conns):
370            fa = pad_to_fp.get(id(c.a))
371            fb = pad_to_fp.get(id(c.b))
372            for fi in {fa, fb}:
373                if fi is not None:
374                    self._fp_conns[fi].append(ci)
375
376    def _rebuild_cache(self) -> None:
377        """Full recompute of the energy cache (init / after a structural change)."""
378        if not self._conns and self.boxed:
379            self._build_index()
380        self._boxes = [self._fp_box(fp) for fp in self.boxed]
381        self._bounds = [b.bounds for b in self._boxes]
382        self._conn_len = [c.est_length for c in self._conns]
383        self._rats = sum(self._conn_len)
384        self._overlap = self._overlap_area(self._boxes) + \
385            self._fixed_text_overlap(self._boxes)
386        self._bbox = self._bbox_from_bounds()
387        self._edge = self._edge_sum_from_bounds(self._bounds)
388        self._containment = self._containment_sum(self._boxes)
389        self._congestion = self._congestion_sum(self._bounds)
390
391    def _congestion_sum(self, bounds) -> float:
392        """Total congestion-field value sampled at the footprint centroids.
393
394        Zero (and skipped) unless ``--place-feedback`` supplied a
395        `router.CongestionField`. Each footprint's body-box centre is sampled and
396        the values summed; minimising this term pushes parts out of the cells where
397        a previous cycle's routing was congested. Like the bbox and edge terms it
398        depends on the absolute layout, so it is an O(N) recompute per move.
399
400        Args:
401            bounds: per-footprint ``(minx, miny, maxx, maxy)`` tuples (e.g.
402                ``self._bounds``).
403
404        Returns:
405            The summed field value (dimensionless), or ``0.0`` when no field.
406        """
407        field = self.p.congestion_field
408        if field is None or not bounds:
409            return 0.0
410        total = 0.0
411        for bx0, by0, bx1, by1 in bounds:
412            total += field.value_at((bx0 + bx1) * 0.5, (by0 + by1) * 0.5)
413        return total
414
415    def _containment_sum(self, boxes) -> float:
416        """Total footprint area protruding outside the kept board outline.
417
418        Zero (and skipped) unless ``--keep-outline`` built a containment polygon
419        (`self._outline_poly`). Penalising the area each box sticks out keeps the
420        placement inside an existing Edge.Cuts instead of regenerating a bounding
421        box. Each box is the buffer-inflated body box, so a small margin to the
422        edge is kept automatically.
423
424        Args:
425            boxes: the per-footprint body boxes (e.g. ``self._boxes``).
426
427        Returns:
428            The summed protruding area (mm²).
429        """
430        poly = self._outline_poly
431        if poly is None:
432            return 0.0
433        return sum(b.difference(poly).area for b in boxes)
434
435    def _containment_touching(self, idxs, boxes) -> float:
436        """Containment-area contribution of just the boxes in ``idxs`` (0 if off)."""
437        if self._outline_poly is None:
438            return 0.0
439        return sum(boxes[i].difference(self._outline_poly).area for i in idxs)
440
441    @staticmethod
442    def _bbox_area(boxes) -> float:
443        """Area of the bounding box enclosing all `boxes` (0 if none)."""
444        if not boxes:
445            return 0.0
446        minx = min(b.bounds[0] for b in boxes)
447        miny = min(b.bounds[1] for b in boxes)
448        maxx = max(b.bounds[2] for b in boxes)
449        maxy = max(b.bounds[3] for b in boxes)
450        return max(0.0, maxx - minx) * max(0.0, maxy - miny)
451
452    def _bbox_from_bounds(self) -> float:
453        """Area of the layout bbox from the cached per-box ``(minx,miny,maxx,maxy)``.
454
455        Reads the plain-tuple bounds cached in ``self._bounds`` rather than calling
456        shapely ``.bounds`` on every box each step, which dominated the placement
457        SA hot loop.
458        """
459        bnds = self._bounds
460        if not bnds:
461            return 0.0
462        minx = min(b[0] for b in bnds)
463        miny = min(b[1] for b in bnds)
464        maxx = max(b[2] for b in bnds)
465        maxy = max(b[3] for b in bnds)
466        return max(0.0, maxx - minx) * max(0.0, maxy - miny)
467
468    def _edge_sum_from_bounds(self, bounds) -> float:
469        """Total edge-affinity distance for the flagged footprints.
470
471        For each flagged footprint, the distance from the reference's target side
472        to the box's *far* side: ``left`` = how far the box's right edge is from
473        the reference's left edge, etc.; ``any`` = the smallest such distance over
474        the four sides. Using the far side means the distance is *gap to the edge
475        + the box's depth perpendicular to it*, so minimising it both pulls the
476        part onto the boundary **and** orients it to lie flat against the edge
477        (long axis parallel) rather than poking inward — otherwise a connector
478        could rotate so only one pad reached the edge. The reference is the kept
479        board outline's bounding box under ``--keep-outline`` (so edge parts snap
480        to the real edge), otherwise the *layout* bounding box (the extent of all
481        ``bounds``). ``0`` when nothing is flagged.
482
483        Args:
484            bounds: per-footprint ``(minx, miny, maxx, maxy)`` tuples, indexed
485                like ``self.boxed`` (e.g. ``self._bounds``).
486
487        Returns:
488            The summed distance (mm).
489        """
490        if not self._flagged or not bounds:
491            return 0.0
492        if self._outline_bounds is not None:
493            minx, miny, maxx, maxy = self._outline_bounds
494        else:
495            minx = min(b[0] for b in bounds)
496            miny = min(b[1] for b in bounds)
497            maxx = max(b[2] for b in bounds)
498            maxy = max(b[3] for b in bounds)
499        total = 0.0
500        for fi, side in self._flagged.items():
501            bx0, by0, bx1, by1 = bounds[fi]
502            if side == "left":
503                d = bx1 - minx
504            elif side == "right":
505                d = maxx - bx0
506            elif side == "top":
507                d = by1 - miny
508            elif side == "bottom":
509                d = maxy - by0
510            else:                                     # "any" — nearest side
511                d = min(bx1 - minx, maxx - bx0, by1 - miny, maxy - by0)
512            total += d
513        return total
514
515    def _cached_energy(self) -> float:
516        """Energy from the current cache (see the module docstring)."""
517        return (self._rats + self.p.overlap_weight * self._overlap
518                + self.p.compact_weight * self._bbox
519                + self.p.edge_weight * self._edge
520                + self.p.containment_weight * self._containment
521                + self.p.congestion_weight * self._congestion)
522
523    def _overlap_touching(self, idxs: set[int], boxes) -> float:
524        """Overlap area of every pair/fixed-text touching a footprint in `idxs`.
525
526        Each footprint-footprint pair with at least one endpoint in `idxs` is
527        counted exactly once (pairs wholly inside `idxs` included once), plus the
528        fixed board-silk-text overlap of the boxes in `idxs`.
529
530        Args:
531            idxs: boxed-indices of the moved footprints.
532            boxes: the per-footprint body boxes to evaluate against.
533
534        Returns:
535            The total overlap (mm^2) attributable to the moved footprints.
536        """
537        tree = STRtree(boxes) if len(boxes) >= 2 else None
538        total = 0.0
539        seen: set[tuple[int, int]] = set()
540        for i in idxs:
541            bi = boxes[i]
542            if tree is not None:
543                for j in tree.query(bi):
544                    j = int(j)
545                    if j == i or not bi.intersects(boxes[j]):
546                        continue
547                    key = (i, j) if i < j else (j, i)
548                    if key in seen:
549                        continue
550                    seen.add(key)
551                    fi, fj = self.boxed[i], self.boxed[j]
552                    if fi.overlap_ok or fj.overlap_ok:
553                        total += self._pad_overlap(fi, fj)
554                    else:
555                        total += bi.intersection(boxes[j]).area
556            if self._fixed_tree is not None and not self.boxed[i].overlap_ok:
557                for j in self._fixed_tree.query(bi):
558                    t = self._fixed_text[int(j)]
559                    if bi.intersects(t):
560                        total += bi.intersection(t).area
561        return total
562
563    def _fp_box(self, fp: Footprint):
564        """Buffer-inflated axis-aligned body box of a footprint.
565
566        Covers pads and any visible silkscreen text (Reference/Value) so the
567        overlap penalty also pushes text labels apart.
568        """
569        hb = self.half_buffer
570        xs0 = min(p.cx - _half_extent(p) for p in fp.pads) - hb
571        ys0 = min(p.cy - _half_extent(p) for p in fp.pads) - hb
572        xs1 = max(p.cx + _half_extent(p) for p in fp.pads) + hb
573        ys1 = max(p.cy + _half_extent(p) for p in fp.pads) + hb
574        # Extend to cover silkscreen text (local coords → board coords).
575        txt_list = self._text_extents.get(id(fp))
576        if txt_list:
577            cos_a = math.cos(math.radians(fp.angle))
578            sin_a = math.sin(math.radians(fp.angle))
579            for lx, ly, hr in txt_list:
580                bx = fp.x + lx * cos_a + ly * sin_a
581                by = fp.y - lx * sin_a + ly * cos_a
582                xs0 = min(xs0, bx - hr)
583                ys0 = min(ys0, by - hr)
584                xs1 = max(xs1, bx + hr)
585                ys1 = max(ys1, by + hr)
586        return box(xs0, ys0, xs1, ys1)
587
588    def _pad_box(self, pad: Pad):
589        """Buffer-inflated axis-aligned box around a single pad at its centre."""
590        he = _half_extent(pad) + self.half_buffer
591        return box(pad.cx - he, pad.cy - he, pad.cx + he, pad.cy + he)
592
593    def _pad_overlap(self, fa: Footprint, fb: Footprint) -> float:
594        """Total intersection area between two footprints' pad boxes (mm²)."""
595        boxes_b = [self._pad_box(p) for p in fb.pads]
596        total = 0.0
597        for a in (self._pad_box(p) for p in fa.pads):
598            for b in boxes_b:
599                if a.intersects(b):
600                    total += a.intersection(b).area
601        return total
602
603    def _overlap_area(self, boxes) -> float:
604        """Penalised overlap across all footprint pairs (mm²).
605
606        Args:
607            boxes: per-footprint body boxes, parallel to ``self.boxed``.
608
609        Returns:
610            Body-box overlap for ordinary pairs, plus pad-only overlap for pairs
611            involving an ``overlap_ok`` footprint.
612        """
613        if len(boxes) < 2:
614            return 0.0
615        tree = STRtree(boxes)
616        total = 0.0
617        for i, bi in enumerate(boxes):
618            for j in tree.query(bi):
619                if j <= i or not bi.intersects(boxes[j]):
620                    continue
621                fi, fj = self.boxed[i], self.boxed[j]
622                if fi.overlap_ok or fj.overlap_ok:
623                    total += self._pad_overlap(fi, fj)
624                else:
625                    total += bi.intersection(boxes[j]).area
626        return total
627
628    def _fixed_text_overlap(self, boxes) -> float:
629        """Overlap (mm²) between footprint boxes and fixed board silk text.
630
631        Args:
632            boxes: per-footprint body boxes, parallel to ``self.boxed``.
633
634        Returns:
635            Total intersection area of each footprint box with the static
636            board-level silkscreen text boxes. ``overlap_ok`` footprints (which
637            are meant to sit over the board) are exempt.
638        """
639        if self._fixed_tree is None:
640            return 0.0
641        total = 0.0
642        for i, bi in enumerate(boxes):
643            if self.boxed[i].overlap_ok:
644                continue
645            for j in self._fixed_tree.query(bi):
646                t = self._fixed_text[j]
647                if bi.intersects(t):
648                    total += bi.intersection(t).area
649        return total
650
651    def _move_delta(self, idxs: set[int]) -> None:
652        """Update the energy cache for a move that touched footprints ``idxs``.
653
654        Recomputes only the parts of the energy the move can change: the lengths
655        of the ratsnest connections incident on the moved footprints, the overlap
656        contributions of the moved boxes (against their neighbours and the fixed
657        board silk text), and the layout bounding box. The connection *topology*
658        and the per-footprint silk extents are fixed for the run, so they are
659        never rebuilt here.
660
661        The cached ``self._boxes`` for ``idxs`` must still hold the *pre-move*
662        boxes when this is called; they are refreshed in place as part of the
663        update. The footprints themselves must already be at their post-move
664        poses (pads synced).
665
666        Args:
667            idxs: boxed-indices of the footprints whose pose changed.
668        """
669        # Ratsnest: only connections incident on a moved footprint change length.
670        seen_conns: set[int] = set()
671        for fi in idxs:
672            for ci in self._fp_conns.get(fi, ()):
673                seen_conns.add(ci)
674        for ci in seen_conns:
675            self._rats -= self._conn_len[ci]
676        # Overlap + containment: remove the moved boxes' old contribution, refresh
677        # the boxes, then add their new contribution. Both are per-box (a box vs
678        # its neighbours / the fixed outline), so only the moved boxes change.
679        self._overlap -= self._overlap_touching(idxs, self._boxes)
680        self._containment -= self._containment_touching(idxs, self._boxes)
681        for fi in idxs:
682            self._boxes[fi] = self._fp_box(self.boxed[fi])
683            self._bounds[fi] = self._boxes[fi].bounds
684        self._overlap += self._overlap_touching(idxs, self._boxes)
685        self._containment += self._containment_touching(idxs, self._boxes)
686        for ci in seen_conns:
687            nl = self._conns[ci].est_length
688            self._conn_len[ci] = nl
689            self._rats += nl
690        # bbox + edge + congestion: cheap O(N) recompute from the cached
691        # plain-tuple bounds (the bbox/edge depend on the global layout extent and
692        # the congestion sample on each centroid, both of which any move can shift).
693        self._bbox = self._bbox_from_bounds()
694        self._edge = self._edge_sum_from_bounds(self._bounds)
695        self._congestion = self._congestion_sum(self._bounds)
696
697    def _energy_components(self) -> tuple[float, float, float]:
698        """Return the ``(ratsnest, overlap_area, bbox_area)`` energy terms."""
699        rats = sum(c.est_length
700                   for c in netlist.build_connections(self.board, self.p.exclude))
701        boxes = [self._fp_box(fp) for fp in self.boxed]
702        overlap = self._overlap_area(boxes) + self._fixed_text_overlap(boxes)
703        if boxes:
704            minx = min(b.bounds[0] for b in boxes)
705            miny = min(b.bounds[1] for b in boxes)
706            maxx = max(b.bounds[2] for b in boxes)
707            maxy = max(b.bounds[3] for b in boxes)
708            bbox_area = max(0.0, maxx - minx) * max(0.0, maxy - miny)
709        else:
710            bbox_area = 0.0
711        return rats, overlap, bbox_area
712
713    def _energy(self) -> float:
714        """Current placement energy (see the module docstring)."""
715        rats, overlap, bbox_area = self._energy_components()
716        boxes = [self._fp_box(fp) for fp in self.boxed]
717        edge = self._edge_sum_from_bounds([b.bounds for b in boxes])
718        containment = self._containment_sum(boxes)
719        congestion = self._congestion_sum([b.bounds for b in boxes])
720        return (rats + self.p.overlap_weight * overlap
721                + self.p.compact_weight * bbox_area + self.p.edge_weight * edge
722                + self.p.containment_weight * containment
723                + self.p.congestion_weight * congestion)
724
725    @staticmethod
726    def _snapshot(fps):
727        """Capture ``(fp, x, y, angle)`` poses for undo/best-tracking."""
728        return [(fp, fp.x, fp.y, fp.angle) for fp in fps]
729
730    @staticmethod
731    def _restore(snap):
732        """Restore poses captured by `_snapshot` and re-sync their pads."""
733        for fp, x, y, angle in snap:
734            fp.x, fp.y, fp.angle = x, y, angle
735            fp.sync_pads()
736
737    def _move(self, temp_frac: float):
738        """Apply one random move and return its undo snapshot and touched indices.
739
740        Args:
741            temp_frac: ``T / t_start`` in ``[t_end/t_start, 1]``; scales the
742                translate step so moves shrink as the schedule cools.
743
744        Returns:
745            ``(snapshot, idxs)`` — the `_snapshot` of the footprints touched (for
746            `_restore`) and the set of their boxed-indices (for `_move_delta`).
747        """
748        r = self.rng.random()
749        if len(self.movable) >= 2 and r < 0.2:
750            a, b = self.rng.sample(self.movable, 2)          # swap origins
751            snap = self._snapshot([a, b])
752            a.x, b.x = b.x, a.x
753            a.y, b.y = b.y, a.y
754            a.sync_pads()
755            b.sync_pads()
756            return snap, {self._idx_of_fp[id(a)], self._idx_of_fp[id(b)]}
757        fp = self.rng.choice(self.movable)
758        snap = self._snapshot([fp])
759        if self.p.rotate_mode != "none" and r < 0.5:         # rotate
760            if self.p.rotate_mode == "free":
761                fp.angle = self.rng.uniform(0.0, 360.0)
762            else:                                            # "ortho"
763                fp.angle = (fp.angle + self.rng.choice((90.0, -90.0, 180.0))) % 360.0
764        else:                                                # translate
765            s = self.p.step * temp_frac
766            fp.x += self.rng.uniform(-s, s)
767            fp.y += self.rng.uniform(-s, s)
768        fp.sync_pads()
769        return snap, {self._idx_of_fp[id(fp)]}
770
771    def run(self, on_progress=None, cancel=None) -> PlaceResult:
772        """Run the annealing loop; leave the board at the best placement seen.
773
774        Args:
775            on_progress: optional callback ``(it, total, energy, best, temp,
776                accept)`` invoked each iteration, where ``accept`` is the fraction
777                of moves accepted over the last ``_ACCEPT_WINDOW`` iterations.
778            cancel: optional `threading.Event`; when set, the loop stops early and
779                the board is left at the best placement found so far (for a GUI
780                Stop button).
781
782        Returns:
783            The `PlaceResult` with start/best energy, run statistics, and the
784            energy breakdown at the best placement.
785        """
786        for fp in self.boxed:
787            fp.sync_pads()
788        if not self.movable:
789            self._rebuild_cache()
790            E = self._cached_energy()
791            return PlaceResult(E, E, 0, 0, 0,
792                               self._rats, self._overlap, self._bbox, self._edge)
793
794        self._rebuild_cache()
795        E = self._cached_energy()
796        start_E = best_E = E
797        best = self._snapshot(self.movable)
798        accepted = 0
799        recent = deque(maxlen=_ACCEPT_WINDOW)   # 1/0 per recent move, for the live ratio
800
801        # Stall detection: count consecutive completed accept-windows whose
802        # acceptance ratio stayed below `stall_ratio`; break after `stall_patience`
803        # of them. Disabled when either knob is non-positive.
804        stall_on = self.p.stall_patience > 0 and self.p.stall_ratio > 0.0
805        stall_count = 0
806        window_seen = 0
807
808        total = self.p.iters if self.p.iters else 1_000_000
809        t0 = time.time()
810        ratio = self.p.t_end / self.p.t_start
811        it = 0
812        while True:
813            if cancel is not None and cancel.is_set():
814                break
815            if self.p.iters is not None and it >= self.p.iters:
816                break
817            if self.p.time_budget is not None and time.time() - t0 >= self.p.time_budget:
818                break
819            if self.p.iters is None and self.p.time_budget is None and it >= 2000:
820                break
821
822            frac = (it / total) if self.p.time_budget is None else min(
823                1.0, (time.time() - t0) / self.p.time_budget)
824            T = self.p.t_start * (ratio ** frac)
825
826            snap, idxs = self._move(T / self.p.t_start)
827            # Snapshot the cache entries the move can disturb, for a cheap revert:
828            # the scalar totals, the moved boxes, and the lengths of the
829            # connections incident on the moved footprints.
830            touched_conns = {ci for i in idxs for ci in self._fp_conns.get(i, ())}
831            cache_save = (self._rats, self._overlap, self._bbox,
832                          self._edge, self._containment,
833                          {i: (self._boxes[i], self._bounds[i]) for i in idxs},
834                          {ci: self._conn_len[ci] for ci in touched_conns})
835            self._move_delta(idxs)
836            E_new = self._cached_energy()
837            dE = E_new - E
838            accept = dE <= 0 or self.rng.random() < math.exp(-dE / max(T, 1e-9))
839            if accept:
840                E = E_new
841                accepted += 1
842                if E < best_E:
843                    best_E = E
844                    best = self._snapshot(self.movable)
845            else:
846                self._restore(snap)
847                (self._rats, self._overlap, self._bbox,
848                 self._edge, self._containment,
849                 saved_boxes, saved_lens) = cache_save
850                for i, (b, bnd) in saved_boxes.items():
851                    self._boxes[i] = b
852                    self._bounds[i] = bnd
853                for ci, ln in saved_lens.items():
854                    self._conn_len[ci] = ln
855            recent.append(1 if accept else 0)
856
857            it += 1
858            window_seen += 1
859            if on_progress is not None:
860                on_progress(it, total, E, best_E, T, sum(recent) / len(recent))
861
862            if stall_on and window_seen >= _ACCEPT_WINDOW:
863                if sum(recent) / len(recent) < self.p.stall_ratio:
864                    stall_count += 1
865                    if stall_count >= self.p.stall_patience:
866                        break
867                else:
868                    stall_count = 0
869                window_seen = 0
870
871        self._restore(best)
872        moved = sum(1 for fp in self.board.footprints if fp.moved)
873        # Recompute the cache at the best placement so the reported breakdown is
874        # exactly consistent with `best_E` (both use the fixed ratsnest topology),
875        # then re-derive best_E from it so they reconcile to the last bit.
876        self._rebuild_cache()
877        best_E = self._cached_energy()
878        return PlaceResult(start_E, best_E, it, accepted, moved,
879                           self._rats, self._overlap, self._bbox, self._edge)

882def recenter(board: Board) -> tuple[float, float]:
883    """Translate the placed footprints back onto their original centroid.
884
885    Placement energy (ratsnest + overlap + bbox-area) is *translation-invariant*:
886    none of its terms depends on where the cluster sits, only on the footprints'
887    relative poses. With nothing locked, the whole group therefore random-walks
888    during annealing and drifts away from the board origin — the more iterations
889    run, the further it migrates. This shifts every movable footprint by a single
890    rigid offset so the movable footprints' centroid returns to where it started,
891    leaving every energy term (and so the placement result) exactly unchanged.
892
893    A rigid translation only restores the original position when the layout is
894    free to move as a whole, so this is a no-op when any footprint is locked:
895    locked footprints are fixed obstacles that anchor the layout in absolute
896    coordinates, so there is no drift to correct and shifting the movable group
897    would only break its alignment with the anchors.
898
899    Args:
900        board: the board whose movable footprints are recentred in place (pads
901            re-synced).
902
903    Returns:
904        The ``(dx, dy)`` offset applied (``(0.0, 0.0)`` when nothing moved, e.g.
905        no movable footprints or any footprint locked).
906    """
907    movable = [fp for fp in board.footprints if fp.pads and not fp.locked]
908    if not movable or any(fp.locked for fp in board.footprints if fp.pads):
909        return 0.0, 0.0
910    n = len(movable)
911    cur_cx = sum(fp.x for fp in movable) / n
912    cur_cy = sum(fp.y for fp in movable) / n
913    orig_cx = sum(fp.x0 for fp in movable) / n
914    orig_cy = sum(fp.y0 for fp in movable) / n
915    dx, dy = orig_cx - cur_cx, orig_cy - cur_cy
916    if abs(dx) < 1e-6 and abs(dy) < 1e-6:
917        return 0.0, 0.0
918    for fp in movable:
919        fp.x += dx
920        fp.y += dy
921        fp.sync_pads()
922    return dx, dy
923

925def place(board: Board, params: PlaceParams | None = None,
926          on_progress=None, runs: int = 1, cancel=None) -> PlaceResult:
927    """Place a board's footprints by simulated annealing; return the best seen.
928
929    Mutates `board`'s footprint poses (and their pads) in place, leaving them at
930    the best placement found. Locked footprints are held fixed; footprints flagged
931    ``Autoroute-overlap`` may overlap others' bodies but not their pads; those
932    flagged ``Autoroute-edge=<side>`` are pulled to the board boundary and oriented
933    flat against it (`edge_weight`). Call
934    `pyautoroute.pcb.apply_placement` afterwards to finalise pad coordinates and
935    regenerate the board outline before routing.
936
937    With ``runs > 1`` the placement is repeated that many times — each restarted
938    from the board's original poses with the seed stepped by the run index — and
939    the lowest-energy placement is kept (best-of-N; SA is stochastic, so several
940    short runs often beat one long run).
941
942    Args:
943        board: the board to place, mutated in place.
944        params: the placement parameters; ``None`` uses defaults.
945        on_progress: optional per-iteration progress callback (see `_Placer.run`).
946        runs: number of independent placement runs; the best is kept.
947        cancel: optional `threading.Event`; when set, stops early (between and
948            within runs) and returns the best placement found so far.
949
950    Returns:
951        The `PlaceResult` with the best placement's energy and run statistics.
952    """
953    params = params or PlaceParams()
954    # keep_outline ties the placement to a fixed outline (absolute coordinates),
955    # so the layout must not be re-centred afterwards — that would shift parts
956    # back out of the outline they were contained within. A congestion field is
957    # likewise absolute (it anchors parts to/away from board cells), so recentring
958    # would invalidate it too.
959    do_recenter = not params.keep_outline and params.congestion_field is None
960    if runs <= 1:
961        result = _Placer(board, params).run(on_progress, cancel)
962        if do_recenter:
963            recenter(board)           # undo translation-invariant drift
964        return result
965
966    orig = [(fp, fp.x, fp.y, fp.angle) for fp in board.footprints]

def recenter(board: pyautoroute.pcb.Board) -> tuple[float, float]: 
883def recenter(board: Board) -> tuple[float, float]:
884    """Translate the placed footprints back onto their original centroid.
885
886    Placement energy (ratsnest + overlap + bbox-area) is *translation-invariant*:
887    none of its terms depends on where the cluster sits, only on the footprints'
888    relative poses. With nothing locked, the whole group therefore random-walks
889    during annealing and drifts away from the board origin — the more iterations
890    run, the further it migrates. This shifts every movable footprint by a single
891    rigid offset so the movable footprints' centroid returns to where it started,
892    leaving every energy term (and so the placement result) exactly unchanged.
893
894    A rigid translation only restores the original position when the layout is
895    free to move as a whole, so this is a no-op when any footprint is locked:
896    locked footprints are fixed obstacles that anchor the layout in absolute
897    coordinates, so there is no drift to correct and shifting the movable group
898    would only break its alignment with the anchors.
899
900    Args:
901        board: the board whose movable footprints are recentred in place (pads
902            re-synced).
903
904    Returns:
905        The ``(dx, dy)`` offset applied (``(0.0, 0.0)`` when nothing moved, e.g.
906        no movable footprints or any footprint locked).
907    """
908    movable = [fp for fp in board.footprints if fp.pads and not fp.locked]
909    if not movable or any(fp.locked for fp in board.footprints if fp.pads):
910        return 0.0, 0.0
911    n = len(movable)
912    cur_cx = sum(fp.x for fp in movable) / n
913    cur_cy = sum(fp.y for fp in movable) / n
914    orig_cx = sum(fp.x0 for fp in movable) / n
915    orig_cy = sum(fp.y0 for fp in movable) / n
916    dx, dy = orig_cx - cur_cx, orig_cy - cur_cy
917    if abs(dx) < 1e-6 and abs(dy) < 1e-6:
918        return 0.0, 0.0
919    for fp in movable:
920        fp.x += dx
921        fp.y += dy
922        fp.sync_pads()
923    return dx, dy
