pyautoroute.router

A* maze router plus an incremental routing state that supports rip-up.

The cost model encodes the optimisation priorities below wirelength: a diagonal step costs its true length (so a 45 degree run beats a 90 degree staircase), a direction change adds a bend penalty (90 degree bends cost more than 45), a layer change adds a via penalty, and B.Cu adds a small per-step penalty so F.Cu wins ties. The heuristic is octile distance to the nearest target (admissible, since every extra cost term is non-negative).

RoutingState layers the dynamic copper (routed tracks/vias) on top of the grid's static occupancy (pads + board edge). It is keyed per connection so two connections of the same net never block each other and any single connection can be ripped up and rerouted exactly — the foundation for simulated annealing.

 79def _turn_units(a: int, b: int) -> int:
 80    """Turn magnitude between two of the 8 compass directions, in 45-deg steps.
 81
 82    Args:
 83        a: previous direction index (0-7), or ``< 0`` for "no previous step".
 84        b: next direction index (0-7).
 85
 86    Returns:

118# --- congestion feedback ------------------------------------------------------
119#
120# A coarse, board-wide "where did routing struggle?" heatmap, fed back into the
121# next cycle's placement (`--place-feedback`) to spread footprints out of the hot
122# zones. It is intentionally low-resolution (a small multiple of the routing
123# pitch) and lives in a *fixed* board frame so successive cycles — whose
124# per-cycle routing grids differ as footprints move — accumulate onto one common
125# raster. Routing itself is untouched: the field is read off the routed results.
126

128@dataclass
129class CongestionField:
130    """A coarse heatmap of routing difficulty over a fixed board frame.
131
132    The board area is rasterised into ``ny × nx`` cells of side ``cell`` mm with
133    origin ``(minx, miny)``; ``values`` (normalised to ``[0, 1]``) is high where
134    routing was hard — dense copper, vias, and the regions of connections left
135    unrouted. Placement samples it at footprint centroids and is pushed toward
136    cooler cells (`pyautoroute.placement`).
137
138    Attributes:
139        minx: frame origin x (mm).
140        miny: frame origin y (mm).
141        cell: cell side (mm).
142        nx: cell columns.
143        ny: cell rows.
144        values: ``(ny, nx)`` float array in ``[0, 1]``; 0 outside is implied
145            (`value_at` returns 0 beyond the frame).
146    """
147    minx: float
148    miny: float
149    cell: float
150    nx: int
151    ny: int
152    values: np.ndarray
153
154    def value_at(self, x: float, y: float) -> float:
155        """Sample the field at board coordinate ``(x, y)``.
156
157        Args:
158            x: board x (mm).
159            y: board y (mm).
160
161        Returns:
162            The cell value (``[0, 1]``), or ``0.0`` outside the frame — a
163            footprint pushed off the field feels no congestion (the placement's
164            compaction/rats-nest terms keep it from drifting away).
165        """
166        c = int((x - self.minx) / self.cell)
167        r = int((y - self.miny) / self.cell)
168        if 0 <= c < self.nx and 0 <= r < self.ny:
169            return float(self.values[r, c])
170        return 0.0
171
172    def blended(self, other: "CongestionField", decay: float) -> "CongestionField":
173        """Exponentially blend an earlier field with a new one on the same frame.
174
175        ``decay`` weights the *accumulated* history: ``decay·self + (1-decay)·other``.
176        Higher ``decay`` remembers longer (signal accumulates), lower reacts faster
177        to the latest cycle. The blend is renormalised so the result stays in
178        ``[0, 1]`` and one runaway cycle can't dominate.
179
180        Args:
181            other: the new field (must share this field's frame).
182            decay: history weight in ``[0, 1]``.
183
184        Returns:
185            A new `CongestionField` on the same frame.
186        """
187        v = decay * self.values + (1.0 - decay) * other.values
188        peak = float(v.max())
189        if peak > 0:
190            v = v / peak
191        return CongestionField(self.minx, self.miny, self.cell, self.nx, self.ny, v)

194def congestion_frame(board, pitch: float, *, cell_mult: int = 4,
195                     margin: float = 5.0) -> CongestionField:
196    """Build an empty `CongestionField` framing a board's pad extent.
197
198    The frame is fixed once (from the freshly loaded board) and reused for every
199    feedback cycle, so each cycle's `congestion_heatmap` rasters onto the same
200    cells regardless of how that cycle's placement (and thus its routing grid)
201    shifted.
202
203    Args:
204        board: the board whose pads define the extent.
205        pitch: the routing-grid pitch (mm); the cell side is ``cell_mult × pitch``.
206        cell_mult: coarse cell side as a multiple of ``pitch`` (default 4).
207        margin: extra border (mm) added on every side so parts that spread
208            outward still land on the field.
209
210    Returns:
211        A zero-valued `CongestionField`; ``nx``/``ny`` are at least 1.
212    """
213    xs = [p.cx for p in board.pads]
214    ys = [p.cy for p in board.pads]
215    if not xs:
216        return CongestionField(0.0, 0.0, max(pitch * cell_mult, 1e-6), 1, 1,
217                               np.zeros((1, 1)))
218    cell = max(pitch * cell_mult, 1e-6)
219    minx, miny = min(xs) - margin, min(ys) - margin
220    maxx, maxy = max(xs) + margin, max(ys) + margin
221    nx = max(1, int(math.ceil((maxx - minx) / cell)))
222    ny = max(1, int(math.ceil((maxy - miny) / cell)))
223    return CongestionField(minx, miny, cell, nx, ny, np.zeros((ny, nx)))

226# relative cell weights for the heatmap (normalised away afterwards, so only
227# their ratios matter): an unrouted connection is the strongest signal, a via
228# next, an ordinary track node the baseline.
229_W_TRACK = 1.0
230_W_VIA = 3.0
231_W_UNROUTED = 10.0

234def congestion_heatmap(conns, results, grid: Grid, frame: CongestionField, *,
235                       blur: float = 1.0) -> CongestionField:
236    """Derive a `CongestionField` from one cycle's routed results.
237
238    Accumulates, onto ``frame``'s cells, the copper density of every routed path
239    (a unit per track node, more per via) plus a strong mark along the straight
240    line between the endpoints of every *unrouted* connection — the two things a
241    spread-out placement could relieve. The raw counts are smoothed with a light
242    Gaussian blur (so the placement sees a gradient, not a step) and normalised to
243    ``[0, 1]``. Routing state is only read, never changed.
244
245    Args:
246        conns: the connection list (parallel to ``results``); supplies the
247            endpoints of unrouted connections.
248        results: per-connection `RouteResult` (``None`` where unrouted).
249        grid: the routing grid the ``results`` were produced on (maps path nodes
250            to board coordinates); may differ frame from ``frame``.
251        frame: the fixed `CongestionField` whose geometry (origin/cell/size) the
252            heatmap is rasterised onto. Its values are ignored.
253        blur: Gaussian-blur sigma in cells (0 disables); softens the field.
254
255    Returns:
256        A new `CongestionField` on ``frame``'s geometry, normalised to ``[0, 1]``.
257    """
258    from scipy.ndimage import gaussian_filter
259
260    acc = np.zeros((frame.ny, frame.nx))
261
262    def _bump(x: float, y: float, w: float) -> None:
263        c = int((x - frame.minx) / frame.cell)
264        r = int((y - frame.miny) / frame.cell)
265        if 0 <= c < frame.nx and 0 <= r < frame.ny:
266            acc[r, c] += w
267
268    for res in results:
269        if res is None:
270            continue
271        prev_layer = None
272        for (layer, col, row) in res.path:
273            x, y = grid.node_xy(col, row)
274            _bump(x, y, _W_TRACK)
275            if prev_layer is not None and layer != prev_layer:
276                _bump(x, y, _W_VIA)            # a via at this node
277            prev_layer = layer
278
279    for res, conn in zip(results, conns):
280        if res is not None:
281            continue
282        ax, ay = conn.a.cx, conn.a.cy
283        bx, by = conn.b.cx, conn.b.cy
284        steps = max(1, int(math.hypot(bx - ax, by - ay) / frame.cell))
285        for s in range(steps + 1):
286            t = s / steps
287            _bump(ax + t * (bx - ax), ay + t * (by - ay), _W_UNROUTED)
288
289    if blur > 0 and acc.any():
290        acc = gaussian_filter(acc, sigma=blur)
291    peak = float(acc.max())
292    if peak > 0:
293        acc = acc / peak
294    return CongestionField(frame.minx, frame.miny, frame.cell,
295                           frame.nx, frame.ny, acc)

300class RoutingState:
301    """Dynamic routed-copper occupancy layered over a grid's static occupancy.
302
303    Occupancy is recorded per connection index, with each connection's net, so
304    rip-up is exact and same-net connections don't block one another.
305    """
306
307    def __init__(self, grid: Grid):
308        """Layer dynamic routed-copper occupancy over a static grid.
309
310        Args:
311            grid: the static routing grid (pad/edge occupancy) to build on.
312        """
313        self.grid = grid
314        self.cover: dict[tuple[int, int, int], set[int]] = {}   # node -> {conn_idx}
315        self.conn_cover: dict[int, set[tuple[int, int, int]]] = {}
316        self.conn_net: dict[int, int] = {}
317        # Vectorised mirror of `cover` for the A* free-mask overlay: per node, the
318        # net id of its committed copper, or `_COVER_EMPTY`/`_COVER_MIXED`. Built
319        # incrementally on commit/ripup (touching only the affected connection's
320        # nodes) so each search overlays "blocked by another net" with one numpy
321        # op instead of a Python loop over every committed node. `cover` /
322        # `conn_net` remain the source of truth; this is a derived index.
323        self.cover_owner = np.full((grid.n_layers, grid.ny, grid.nx),
324                                   _COVER_EMPTY, dtype=np.int32)
325        # hfield (octile heuristic) cache, keyed by the connection's target set;
326        # a connection's targets are fixed, so its field is identical every reroute.
327        self._hfield_cache: dict[frozenset, np.ndarray] = {}
328
329    def _refresh_owner(self, node: tuple[int, int, int]) -> None:
330        """Recompute `cover_owner[node]` from the connections currently covering it."""
331        occ = self.cover.get(node)
332        li, c, r = node
333        if not occ:
334            self.cover_owner[li, r, c] = _COVER_EMPTY
335            return
336        it = iter(occ)
337        net = self.conn_net[next(it)]
338        for i in it:
339            if self.conn_net[i] != net:
340                self.cover_owner[li, r, c] = _COVER_MIXED
341                return
342        self.cover_owner[li, r, c] = net
343
344    def is_free(self, layer_idx: int, col: int, row: int, net_id: int) -> bool:
345        """Whether `net_id` may occupy a node, considering routed copper too.
346
347        Args:
348            layer_idx: copper layer index.
349            col: node column.
350            row: node row.
351            net_id: the routing net's id.
352
353        Returns:
354            True if the static grid permits it and no *other* net's committed
355            copper covers the node.
356        """
357        if not self.grid.is_free(layer_idx, col, row, net_id):
358            return False
359        occ = self.cover.get((layer_idx, col, row))
360        if occ:
361            for idx in occ:
362                if self.conn_net[idx] != net_id:
363                    return False
364        return True
365
366    def can_via(self, col: int, row: int, net_id: int) -> bool:
367        """Whether a via for `net_id` fits at a node, over static + routed copper.
368
369        Args:
370            col: node column for the via centre.
371            row: node row for the via centre.
372            net_id: the routing net's id.
373
374        Returns:
375            True if every node in the via-clearance stencil is free for `net_id`
376            on all copper layers.
377        """
378        for dc, dr in self.grid._via_stencil:
379            c, r = col + dc, row + dr
380            if not self.grid.in_bounds(c, r):
381                return False
382            for li in range(self.grid.n_layers):
383                if not self.is_free(li, c, r, net_id):
384                    return False
385        return True
386
387    def commit(self, conn_idx: int, result: RouteResult) -> None:
388        """Record a routed connection's (inflated) copper as occupancy.
389
390        Args:
391            conn_idx: the connection's index, used as the rip-up key.
392            result: the routed path to commit.
393        """
394        net_id = self.grid.net_id(result.net)
395        nodes = self._covered_nodes(result)
396        self.conn_net[conn_idx] = net_id
397        self.conn_cover[conn_idx] = nodes
398        for nd in nodes:
399            self.cover.setdefault(nd, set()).add(conn_idx)
400            self._refresh_owner(nd)
401
402    def ripup(self, conn_idx: int) -> None:
403        """Remove a connection's committed copper from the occupancy.
404
405        Args:
406            conn_idx: the connection index previously passed to `commit`.
407        """
408        for nd in self.conn_cover.pop(conn_idx, ()):
409            occ = self.cover.get(nd)
410            if occ is not None:
411                occ.discard(conn_idx)
412                if not occ:
413                    del self.cover[nd]
414            self._refresh_owner(nd)
415        self.conn_net.pop(conn_idx, None)
416
417    def _raster(self, geom, layer_idx: int, out: set):
418        """Add every grid node whose centre lies inside `geom` to `out`.
419
420        Args:
421            geom: the shapely area to rasterise.
422            layer_idx: the layer index tagged onto each added node.
423            out: the ``(layer, col, row)`` node set extended in place.
424        """
425        m = self.grid._mesh_in_bbox(geom.bounds)
426        if m is None:
427            return
428        cols, rows, xx, yy = m
429        mask = shapely.contains_xy(geom, xx, yy)
430        if not mask.any():
431            return
432        rr, cc = np.where(mask)
433        for k in range(len(rr)):
434            out.add((layer_idx, int(cols[cc[k]]), int(rows[rr[k]])))
435
436    def _covered_nodes(self, result: RouteResult) -> set:
437        """Grid nodes a routed path occupies, inflated by the clearance margin.
438
439        Args:
440            result: the routed path.
441
442        Returns:
443            The ``(layer, col, row)`` nodes the path's copper (tracks + via
444            disks) claims for clearance bookkeeping.
445        """
446        grid = self.grid
447        width = grid.rules.track_width_for(result.net)
448        via_r = grid.rules.via_diameter_for(result.net) / 2.0
449        # Treat committed copper like a pad obstacle: grow it by the full grid
450        # margin (= routing-track half-width + clearance + discretisation safety).
451        # The routing-track half-width term matters on multi-class boards where a
452        # later wide track would otherwise encroach on this one's clearance.
453        extra = grid.margin
454        nodes: set = set(result.path)
455        for (l0, c0, r0), (l1, c1, r1) in zip(result.path, result.path[1:]):
456            if l0 != l1:
457                x, y = grid.node_xy(c0, r0)
458                disk = Point(x, y).buffer(via_r + extra)
459                for li in range(grid.n_layers):
460                    self._raster(disk, li, nodes)
461            else:
462                x0, y0 = grid.node_xy(c0, r0)
463                x1, y1 = grid.node_xy(c1, r1)
464                track = LineString([(x0, y0), (x1, y1)]).buffer(width / 2.0 + extra)
465                self._raster(track, l0, nodes)
466        return nodes

471def astar(state: RoutingState, net_id: int,
472          sources: list[tuple[int, int, int]],
473          targets: list[tuple[int, int, int]],
474          params: RouteParams | None = None) -> list[tuple[int, int, int]] | None:
475    """Find a min-cost node path from any source to any target.
476
477    Search internals are optimised for the Python inner loop without changing
478    the cost model or results: search states are packed into a single integer
479    key (``((layer*ny + row)*nx + col)*9 + dir+1``) instead of a tuple; a per-net
480    boolean free mask is built once from the static occupancy so neighbour
481    freeness is a direct array index rather than a `RoutingState.is_free` call;
482    the octile heuristic is precomputed over the whole grid as a numpy field; and
483    the via-target layer list is cached on the grid. These are transparent — the
484    returned path is identical to the unoptimised search.
485
486    When ``params.search_margin`` is set, the search is bounded to a box around
487    the source/target nodes (grown by that margin in mm), masking everything
488    outside it not-free. The box widens and the search retries on failure,
489    falling back to the full grid, so a route is still found whenever one exists;
490    the trade-off is that a bounded route may be slightly longer than the global
491    optimum. With ``search_margin=None`` (the default) the whole grid is searched.
492
493    Args:
494        state: the live routing state (occupancy the search must respect).
495        net_id: the routing net's id (nodes owned by other nets are blocked).
496        sources: candidate start nodes ``(layer, col, row)`` (a pad's accesses).
497        targets: candidate goal nodes ``(layer, col, row)``.
498        params: cost-model parameters; defaults are used when `None`.
499
500    Returns:
501        The ``(layer, col, row)`` path from a source to a target, or `None` if
502        none is reachable within the expansion budget.
503    """
504    params = params or RouteParams()
505    grid = state.grid
506    if not sources or not targets:
507        return None
508    nx, ny, n_layers, pitch = grid.nx, grid.ny, grid.n_layers, grid.pitch
509
510    # --- flattened integer state key ----------------------------------------
511    # `(layer, col, row, dir)` is packed into a single int; dict lookups on ints
512    # are faster than on tuples and avoid a tuple allocation per neighbour. `dir`
513    # ranges over -1..7, shifted to 0..8 by the +1 so all keys stay non-negative.
514    #   key = ((layer * ny + row) * nx + col) * 9 + (dir + 1)
515    _D = 9
516    _LROW = nx * _D
517
518    def encode(li: int, c: int, r: int, d: int) -> int:
519        return (li * ny + r) * _LROW + c * _D + (d + 1)
520
521    # --- per-net free mask + heuristic field --------------------------------
522    # These two numpy precomputes dominate the per-call cost, so they are built
523    # by `_precompute` for a given box and (when the search is bounded) only
524    # within that box. Outside the box every node is not-free and the heuristic
525    # is +inf, which the search never reaches — so a bounded search pays for the
526    # box area rather than the whole grid.
527    #   * free mask: a node is free for this net iff the static grid permits it
528    #     (FREE or owned by this net) AND no *other* net's committed copper covers
529    #     it; indexing the boolean array directly avoids a per-expansion call.
530    #   * heuristic field: octile distance to the nearest target × pitch (layer-
531    #     and direction-independent, so a single numpy broadcast per target).
532    owner = grid.owner
533    target_set = set((li, c, r) for (li, c, r) in targets)
534    tgt_cr = [(c, r) for (_, c, r) in targets]
535
536    def _precompute(cmin: int, cmax: int, rmin: int, rmax: int):
537        cs_, rs_ = slice(cmin, cmax + 1), slice(rmin, rmax + 1)
538        free = np.zeros((n_layers, ny, nx), dtype=bool)
539        ow = owner[:, rs_, cs_]
540        free[:, rs_, cs_] = (ow == FREE) | (ow == net_id)
541        # Overlay dynamic routed copper: a node is blocked iff another net (or a
542        # mix of nets) covers it. `cover_owner` is maintained incrementally, so
543        # this is one vectorised numpy op instead of a Python loop over every
544        # committed node — the dominant cost of a reroute during annealing.
545        co = state.cover_owner[:, rs_, cs_]
546        free[:, rs_, cs_] &= (co == _COVER_EMPTY) | (co == net_id)
547
548        # Heuristic field. A connection's targets are fixed, so for the full-grid
549        # (unbounded) search the field is identical every reroute — cache it.
550        full = cmin == 0 and cmax == nx - 1 and rmin == 0 and rmax == ny - 1
551        if full:
552            key = frozenset(tgt_cr)
553            cached = state._hfield_cache.get(key)
554            if cached is not None:
555                return free, cached
556        hfield = np.full((ny, nx), np.inf)
557        bcols = np.arange(cmin, cmax + 1)[None, :]
558        brows = np.arange(rmin, rmax + 1)[:, None]
559        box_h = np.full((rmax - rmin + 1, cmax - cmin + 1), np.inf)
560        for tc, tr in tgt_cr:
561            dx = np.abs(bcols - tc)
562            dy = np.abs(brows - tr)
563            lo = np.minimum(dx, dy)
564            hi = np.maximum(dx, dy)
565            box_h = np.minimum(box_h, (hi - lo) + lo * SQRT2)
566        hfield[rs_, cs_] = box_h * pitch
567        if full:
568            state._hfield_cache[key] = hfield
569        return free, hfield
570
571    # --- via neighbourhood (constant per grid) ------------------------------
572    # The set of layers a via can jump to from any node is just "all other
573    # layers"; precompute it once on the grid so the via branch doesn't rebuild
574    # the layer list on every expansion.
575    via_layers = getattr(grid, "_via_layer_neighbours", None)
576    if via_layers is None:
577        via_layers = [tuple(j for j in range(n_layers) if j != i)
578                      for i in range(n_layers)]
579        grid._via_layer_neighbours = via_layers
580
581    def _search(free, hfield) -> list[tuple[int, int, int]] | None:
582        """Run one A* search over the given per-net `free` mask + heuristic field.
583
584        These two arrays are all that vary between a bounded attempt and the
585        full-grid fallback, so the whole search (both the Cython fast path and the
586        pure-Python heap loop) takes them as parameters. Out-of-box nodes are
587        not-free with a +inf heuristic, which bounds the frontier without any
588        other change.
589        """
590        def is_free(li: int, c: int, r: int) -> bool:
591            return 0 <= c < nx and 0 <= r < ny and bool(free[li, r, c])
592
593        def h(c: int, r: int) -> float:
594            return float(hfield[r, c])
595
596        def can_via(c: int, r: int) -> bool:
597            for dc, dr in grid._via_stencil:
598                cc, rr = c + dc, r + dr
599                if not (0 <= cc < nx and 0 <= rr < ny):
600                    return False
601                if not free[:, rr, cc].all():
602                    return False
603            return True
604
605        # --- optional Cython fast path --------------------------------------
606        # The native core consumes the exact same precomputed structures (free
607        # mask, heuristic field, via stencil/layer lists) and integer state
608        # packing, so it returns a bit-for-bit identical path. It is skipped
609        # transparently when the extension is not built (`_USE_C_ASTAR` is False).
610        if _USE_C_ASTAR:
611            free_c = np.ascontiguousarray(free, dtype=np.uint8)
612            hfield_c = np.ascontiguousarray(hfield, dtype=np.float64)
613            return _astar_fast(
614                free_c, hfield_c,
615                list(sources), list(targets),
616                [(int(dc), int(dr)) for dc, dr in grid._via_stencil],
617                [tuple(v) for v in via_layers],
618                float(pitch), float(params.via_cost),
619                float(params.bend45), float(params.bend90),
620                float(params.bend135), float(params.bend180),
621                float(params.back_layer_penalty),
622                int(params.max_expansions),
623            )
624
625        counter = itertools.count()
626        gscore: dict[int, float] = {}
627        # key -> (predecessor_key, (layer, col, row)); the node tuple is stored so
628        # path reconstruction needs no decode of the packed integer key.
629        came: dict[int, tuple | None] = {}
630        heap = []
631        for (li, c, r) in sources:
632            if not is_free(li, c, r):
633                continue
634            s = encode(li, c, r, -1)
635            gscore[s] = 0.0
636            came[s] = (None, (li, c, r))
637            heapq.heappush(heap, (h(c, r), next(counter), s, li, c, r, -1))
638
639        expansions = 0
640        while heap:
641            f, _, st, li, c, r, pdir = heapq.heappop(heap)
642            g = gscore[st]
643            if f > g + h(c, r) + 1e-9:
644                continue
645            if (li, c, r) in target_set:
646                return _reconstruct(came, st)
647
648            expansions += 1
649            if expansions > params.max_expansions:
650                return None
651
652            for di, (dx, dy) in enumerate(_DIRS):
653                nc, nr = c + dx, r + dy
654                if not is_free(li, nc, nr):
655                    continue
656                diagonal = dx != 0 and dy != 0
657                if diagonal and not (is_free(li, c + dx, r)
658                                     and is_free(li, c, r + dy)):
659                    continue
660                step = pitch * (SQRT2 if diagonal else 1.0)
661                cost = step + params.bend(_turn_units(pdir, di))
662                if li != 0:
663                    cost += params.back_layer_penalty
664                ng = g + cost
665                ns = encode(li, nc, nr, di)
666                if ng < gscore.get(ns, math.inf):
667                    gscore[ns] = ng
668                    came[ns] = (st, (li, nc, nr))
669                    heapq.heappush(heap, (ng + h(nc, nr), next(counter),
670                                          ns, li, nc, nr, di))
671
672            if n_layers > 1 and can_via(c, r):
673                ng = g + params.via_cost
674                for nli in via_layers[li]:
675                    ns = encode(nli, c, r, -1)
676                    if ng < gscore.get(ns, math.inf):
677                        gscore[ns] = ng
678                        came[ns] = (st, (nli, c, r))
679                        heapq.heappush(heap, (ng + h(c, r), next(counter),
680                                              ns, nli, c, r, -1))
681
682        return None
683
684    # --- bounded search with widen-on-failure -------------------------------
685    # Unbounded: precompute and search the whole grid (the historical behaviour).
686    if params.search_margin is None:
687        return _search(*_precompute(0, nx - 1, 0, ny - 1))
688
689    # Bound the search to a box around the source/target nodes grown by the
690    # margin. `_precompute` fills the free mask and heuristic field only inside
691    # the box, so each bounded attempt pays for the box area, not the whole grid.
692    # On failure widen the box and retry; once it covers the grid this is the
693    # unbounded search, so a route is still found whenever one exists.
694    cs = [c for (_, c, _) in sources] + [c for (_, c, _) in targets]
695    rs = [r for (_, _, r) in sources] + [r for (_, _, r) in targets]
696    cmin0, cmax0 = min(cs), max(cs)
697    rmin0, rmax0 = min(rs), max(rs)
698    pad = max(1, int(math.ceil(params.search_margin / pitch)))
699    while True:
700        cmin, cmax = max(0, cmin0 - pad), min(nx - 1, cmax0 + pad)
701        rmin, rmax = max(0, rmin0 - pad), min(ny - 1, rmax0 + pad)
702        full = cmin == 0 and cmax == nx - 1 and rmin == 0 and rmax == ny - 1
703        path = _search(*_precompute(cmin, cmax, rmin, rmax))
704        if path is not None or full:
705            return path
706        pad *= 2

709def _reconstruct(came, st) -> list[tuple[int, int, int]]:
710    """Rebuild the node path from the A* came-from map.
711
712    Args:
713        came: mapping of integer search-state key -> ``(predecessor_key,
714            (layer, col, row))`` (or `None` at a source).
715        st: the goal search-state key to walk back from.
716
717    Returns:
718        The ``(layer, col, row)`` path from source to goal, with consecutive
719        duplicates removed.
720    """
721    out = []
722    while st is not None:
723        prev, node = came[st]
724        out.append(node)
725        st = prev
726    out.reverse()
727    dedup = [out[0]]
728    for n in out[1:]:
729        if n != dedup[-1]:
730            dedup.append(n)
731    return dedup

734def route_connection(state: RoutingState, net: str,
735                     sources: list[tuple[int, int, int]],
736                     targets: list[tuple[int, int, int]],
737                     params: RouteParams | None = None,
738                     src_xy: tuple[float, float] | None = None,
739                     dst_xy: tuple[float, float] | None = None) -> RouteResult | None:
740    """Route one connection and package the path with its metrics.
741
742    Args:
743        state: the live routing state (occupancy).
744        net: the connection's net name.
745        sources: candidate start nodes (one pad's access nodes).
746        targets: candidate goal nodes (the other pad's access nodes).
747        params: cost-model parameters; defaults are used when `None`.
748        src_xy: the source pad's anchor (centre) coordinates, if known; recorded
749            on the result so the writer stubs the track to the pad anchor.
750        dst_xy: the target pad's anchor (centre) coordinates, if known.
751
752    Returns:
753        The `RouteResult` (path + length + vias), or `None` if unroutable.
754    """
755    net_id = state.grid.net_id(net)
756    path = astar(state, net_id, sources, targets, params)
757    if path is None:
758        return None
759    length, vias = _path_metrics(state.grid, path)
760    length += _stub_length(state.grid, path, src_xy, dst_xy)
761    return RouteResult(net=net, path=path, length=length, vias=vias,
762                       src_xy=src_xy, dst_xy=dst_xy)

765def _stub_length(grid: Grid, path, src_xy, dst_xy) -> float:
766    """Extra wirelength of the pad-anchor stubs at the path's two ends.
767
768    Args:
769        grid: the grid (node -> coordinate conversion).
770        path: the routed ``(layer, col, row)`` node path.
771        src_xy: the source pad anchor, or `None`.
772        dst_xy: the target pad anchor, or `None`.
773
774    Returns:
775        The summed length (mm) of the (non-degenerate) endpoint stubs.
776    """
777    extra = 0.0
778    for xy, node in ((src_xy, path[0]), (dst_xy, path[-1])):
779        if xy is None:
780            continue
781        nx, ny = grid.node_xy(node[1], node[2])
782        d = math.hypot(nx - xy[0], ny - xy[1])
783        if d >= _STUB_EPS:
784            extra += d
785    return extra

788def _path_metrics(grid: Grid, path) -> tuple[float, int]:
789    """Measure a node path's wirelength and via count.
790
791    Args:
792        grid: the grid (for the pitch / coordinate scale).
793        path: the ``(layer, col, row)`` node path.
794
795    Returns:
796        ``(length_mm, n_vias)`` — layer changes count as vias, in-layer steps
797        add their geometric length.
798    """
799    length = 0.0
800    vias = 0
801    for (l0, c0, r0), (l1, c1, r1) in zip(path, path[1:]):
802        if l0 != l1:
803            vias += 1
804        else:
805            length += math.hypot(c1 - c0, r1 - r0) * grid.pitch
806    return length, vias

811def route_all(state: RoutingState, connections, order: list[int],
812              params: RouteParams | None = None,
813              on_progress=None, on_partial=None) -> BoardRouting:
814    """Route connections in the given order, committing each success to `state`.
815
816    Args:
817        state: the routing state to commit successful routes into.
818        connections: the connections to route (indexed by `order`).
819        order: indices into `connections` giving the routing order.
820        params: cost-model parameters; defaults are used when `None`.
821        on_progress: optional callback ``(done, total, routed, unrouted)`` after
822            each connection.
823        on_partial: optional callback ``(conn_idx, result_or_none)`` called after
824            each connection, for callers that need the partial results list
825            as it builds up (e.g. live board canvas updates).
826
827    Returns:
828        A `BoardRouting` with per-connection results and aggregate metrics.
829    """
830    results: list[RouteResult | None] = [None] * len(connections)
831    routed = unrouted = 0
832    total_len = 0.0
833    total_vias = 0
834    for k, idx in enumerate(order):
835        conn = connections[idx]
836        srcs = state.grid.pad_access_nodes(conn.a)
837        tgts = state.grid.pad_access_nodes(conn.b)
838        res = route_connection(state, conn.net, srcs, tgts, params,
839                               src_xy=(conn.a.cx, conn.a.cy),
840                               dst_xy=(conn.b.cx, conn.b.cy))
841        results[idx] = res
842        if res is not None:
843            state.commit(idx, res)
844            routed += 1
845            total_len += res.length
846            total_vias += res.vias
847        else:
848            unrouted += 1
849        if on_partial is not None:
850            on_partial(idx, res)
851        if on_progress is not None:
852            on_progress(k + 1, len(order), routed, unrouted)
853    return BoardRouting(results, routed, unrouted, total_len, total_vias)

858def path_to_nodes(board, grid: Grid, result: RouteResult) -> list[SList]:
859    """Convert a routed node path into KiCad ``(segment ...)`` / ``(via ...)`` nodes.
860
861    Collinear runs are merged into single segments and layer changes become vias.
862
863    Args:
864        board: the board (net-reference style for the new nodes).
865        grid: the grid (node -> coordinate conversion, layer names).
866        result: the routed path to serialise.
867
868    Returns:
869        The s-expression nodes to append to the routed board.
870    """
871    from .pcb import make_via
872
873    path = result.path
874    nodes: list[SList] = []
875    if len(path) < 2:
876        return nodes
877
878    rules = grid.rules
879    width = rules.track_width_for(result.net)
880    via_d = rules.via_diameter_for(result.net)
881    via_drill = rules.via_drill_for(result.net)
882
883    run_start = path[0]
884    prev = path[0]
885    prev_dir = None
886    for cur in path[1:]:
887        l0, c0, r0 = prev
888        l1, c1, r1 = cur
889        if l0 != l1:
890            if run_start[1:] != prev[1:]:
891                nodes.append(_seg(board, grid, result.net, run_start, prev, width))
892            x, y = grid.node_xy(c0, r0)
893            nodes.append(make_via(board, x, y, via_d, via_drill,
894                                  grid.layers[l0], grid.layers[l1], result.net))
895            run_start = cur
896            prev_dir = None
897        else:
898            d = (_sign(c1 - c0), _sign(r1 - r0))
899            if prev_dir is not None and d != prev_dir:
900                nodes.append(_seg(board, grid, result.net, run_start, prev, width))
901                run_start = prev
902            prev_dir = d
903        prev = cur
904
905    if run_start[1:] != prev[1:] and run_start[0] == prev[0]:
906        nodes.append(_seg(board, grid, result.net, run_start, prev, width))
907
908    # Stub each end to the pad anchor so the track terminates on the pad centre.
909    # Each stub runs from a terminal grid node (inside the pad) to the pad's
910    # centre (also inside the pad), so it stays within the pad's own copper and
911    # adds no clearance violation.
912    for xy, node in ((result.src_xy, path[0]), (result.dst_xy, path[-1])):
913        stub = _centre_stub(board, grid, result.net, xy, node, width)
914        if stub is not None:
915            nodes.append(stub)
916    return nodes

132

156        """Sample the field at board coordinate ``(x, y)``.
157

159            x: board x (mm).

235def congestion_heatmap(conns, results, grid: Grid, frame: CongestionField, *,
236                       blur: float = 1.0) -> CongestionField:
237    """Derive a `CongestionField` from one cycle's routed results.
238
239    Accumulates, onto ``frame``'s cells, the copper density of every routed path
240    (a unit per track node, more per via) plus a strong mark along the straight
241    line between the endpoints of every *unrouted* connection — the two things a
242    spread-out placement could relieve. The raw counts are smoothed with a light
243    Gaussian blur (so the placement sees a gradient, not a step) and normalised to
244    ``[0, 1]``. Routing state is only read, never changed.
245
246    Args:
247        conns: the connection list (parallel to ``results``); supplies the
248            endpoints of unrouted connections.
249        results: per-connection `RouteResult` (``None`` where unrouted).
250        grid: the routing grid the ``results`` were produced on (maps path nodes
251            to board coordinates); may differ frame from ``frame``.
252        frame: the fixed `CongestionField` whose geometry (origin/cell/size) the
253            heatmap is rasterised onto. Its values are ignored.
254        blur: Gaussian-blur sigma in cells (0 disables); softens the field.
255
256    Returns:
257        A new `CongestionField` on ``frame``'s geometry, normalised to ``[0, 1]``.
258    """

260
261    acc = np.zeros((frame.ny, frame.nx))
262
263    def _bump(x: float, y: float, w: float) -> None:
264        c = int((x - frame.minx) / frame.cell)
265        r = int((y - frame.miny) / frame.cell)
266        if 0 <= c < frame.nx and 0 <= r < frame.ny:
267            acc[r, c] += w
268
269    for res in results:
270        if res is None:
271            continue
272        prev_layer = None
273        for (layer, col, row) in res.path:
274            x, y = grid.node_xy(col, row)
275            _bump(x, y, _W_TRACK)
276            if prev_layer is not None and layer != prev_layer:
277                _bump(x, y, _W_VIA)            # a via at this node
278            prev_layer = layer

507    if not sources or not targets:
508        return None
509    nx, ny, n_layers, pitch = grid.nx, grid.ny, grid.n_layers, grid.pitch
510
511    # --- flattened integer state key ----------------------------------------
512    # `(layer, col, row, dir)` is packed into a single int; dict lookups on ints
513    # are faster than on tuples and avoid a tuple allocation per neighbour. `dir`
514    # ranges over -1..7, shifted to 0..8 by the +1 so all keys stay non-negative.
515    #   key = ((layer * ny + row) * nx + col) * 9 + (dir + 1)
516    _D = 9
517    _LROW = nx * _D
518
519    def encode(li: int, c: int, r: int, d: int) -> int:
520        return (li * ny + r) * _LROW + c * _D + (d + 1)
521
522    # --- per-net free mask + heuristic field --------------------------------
523    # These two numpy precomputes dominate the per-call cost, so they are built
524    # by `_precompute` for a given box and (when the search is bounded) only
525    # within that box. Outside the box every node is not-free and the heuristic
526    # is +inf, which the search never reaches — so a bounded search pays for the
527    # box area rather than the whole grid.
528    #   * free mask: a node is free for this net iff the static grid permits it
529    #     (FREE or owned by this net) AND no *other* net's committed copper covers
530    #     it; indexing the boolean array directly avoids a per-expansion call.
531    #   * heuristic field: octile distance to the nearest target × pitch (layer-
532    #     and direction-independent, so a single numpy broadcast per target).
533    owner = grid.owner
534    target_set = set((li, c, r) for (li, c, r) in targets)
535    tgt_cr = [(c, r) for (_, c, r) in targets]
536
537    def _precompute(cmin: int, cmax: int, rmin: int, rmax: int):
538        cs_, rs_ = slice(cmin, cmax + 1), slice(rmin, rmax + 1)
539        free = np.zeros((n_layers, ny, nx), dtype=bool)
540        ow = owner[:, rs_, cs_]
541        free[:, rs_, cs_] = (ow == FREE) | (ow == net_id)
542        # Overlay dynamic routed copper: a node is blocked iff another net (or a
543        # mix of nets) covers it. `cover_owner` is maintained incrementally, so
544        # this is one vectorised numpy op instead of a Python loop over every
545        # committed node — the dominant cost of a reroute during annealing.
546        co = state.cover_owner[:, rs_, cs_]
547        free[:, rs_, cs_] &= (co == _COVER_EMPTY) | (co == net_id)
548
549        # Heuristic field. A connection's targets are fixed, so for the full-grid
550        # (unbounded) search the field is identical every reroute — cache it.
551        full = cmin == 0 and cmax == nx - 1 and rmin == 0 and rmax == ny - 1
552        if full:
553            key = frozenset(tgt_cr)
554            cached = state._hfield_cache.get(key)
555            if cached is not None:
556                return free, cached
557        hfield = np.full((ny, nx), np.inf)
558        bcols = np.arange(cmin, cmax + 1)[None, :]
559        brows = np.arange(rmin, rmax + 1)[:, None]
560        box_h = np.full((rmax - rmin + 1, cmax - cmin + 1), np.inf)
561        for tc, tr in tgt_cr:
562            dx = np.abs(bcols - tc)
563            dy = np.abs(brows - tr)
564            lo = np.minimum(dx, dy)
565            hi = np.maximum(dx, dy)
566            box_h = np.minimum(box_h, (hi - lo) + lo * SQRT2)
567        hfield[rs_, cs_] = box_h * pitch
568        if full:
569            state._hfield_cache[key] = hfield
570        return free, hfield
571
572    # --- via neighbourhood (constant per grid) ------------------------------
573    # The set of layers a via can jump to from any node is just "all other
574    # layers"; precompute it once on the grid so the via branch doesn't rebuild
575    # the layer list on every expansion.
576    via_layers = getattr(grid, "_via_layer_neighbours", None)
577    if via_layers is None:
578        via_layers = [tuple(j for j in range(n_layers) if j != i)
579                      for i in range(n_layers)]
580        grid._via_layer_neighbours = via_layers
581
582    def _search(free, hfield) -> list[tuple[int, int, int]] | None:
583        """Run one A* search over the given per-net `free` mask + heuristic field.
584
585        These two arrays are all that vary between a bounded attempt and the
586        full-grid fallback, so the whole search (both the Cython fast path and the
587        pure-Python heap loop) takes them as parameters. Out-of-box nodes are
588        not-free with a +inf heuristic, which bounds the frontier without any
589        other change.
590        """
591        def is_free(li: int, c: int, r: int) -> bool:
592            return 0 <= c < nx and 0 <= r < ny and bool(free[li, r, c])
593
594        def h(c: int, r: int) -> float:
595            return float(hfield[r, c])
596
597        def can_via(c: int, r: int) -> bool:
598            for dc, dr in grid._via_stencil:
599                cc, rr = c + dc, r + dr
600                if not (0 <= cc < nx and 0 <= rr < ny):
601                    return False
602                if not free[:, rr, cc].all():
603                    return False
604            return True
605
606        # --- optional Cython fast path --------------------------------------
607        # The native core consumes the exact same precomputed structures (free
608        # mask, heuristic field, via stencil/layer lists) and integer state
609        # packing, so it returns a bit-for-bit identical path. It is skipped
610        # transparently when the extension is not built (`_USE_C_ASTAR` is False).
611        if _USE_C_ASTAR:
612            free_c = np.ascontiguousarray(free, dtype=np.uint8)
613            hfield_c = np.ascontiguousarray(hfield, dtype=np.float64)
614            return _astar_fast(
615                free_c, hfield_c,
616                list(sources), list(targets),
617                [(int(dc), int(dr)) for dc, dr in grid._via_stencil],
618                [tuple(v) for v in via_layers],
619                float(pitch), float(params.via_cost),
620                float(params.bend45), float(params.bend90),
621                float(params.bend135), float(params.bend180),
622                float(params.back_layer_penalty),
623                int(params.max_expansions),
624            )
625
626        counter = itertools.count()
627        gscore: dict[int, float] = {}
628        # key -> (predecessor_key, (layer, col, row)); the node tuple is stored so
629        # path reconstruction needs no decode of the packed integer key.
630        came: dict[int, tuple | None] = {}
631        heap = []
632        for (li, c, r) in sources:
633            if not is_free(li, c, r):
634                continue
635            s = encode(li, c, r, -1)
636            gscore[s] = 0.0
637            came[s] = (None, (li, c, r))
638            heapq.heappush(heap, (h(c, r), next(counter), s, li, c, r, -1))
639
640        expansions = 0
641        while heap:
642            f, _, st, li, c, r, pdir = heapq.heappop(heap)
643            g = gscore[st]
644            if f > g + h(c, r) + 1e-9:
645                continue
646            if (li, c, r) in target_set:
647                return _reconstruct(came, st)
648
649            expansions += 1
650            if expansions > params.max_expansions:
651                return None
652
653            for di, (dx, dy) in enumerate(_DIRS):
654                nc, nr = c + dx, r + dy
655                if not is_free(li, nc, nr):
656                    continue
657                diagonal = dx != 0 and dy != 0
658                if diagonal and not (is_free(li, c + dx, r)
659                                     and is_free(li, c, r + dy)):
660                    continue
661                step = pitch * (SQRT2 if diagonal else 1.0)
662                cost = step + params.bend(_turn_units(pdir, di))
663                if li != 0:
664                    cost += params.back_layer_penalty
665                ng = g + cost
666                ns = encode(li, nc, nr, di)
667                if ng < gscore.get(ns, math.inf):
668                    gscore[ns] = ng
669                    came[ns] = (st, (li, nc, nr))
670                    heapq.heappush(heap, (ng + h(nc, nr), next(counter),
671                                          ns, li, nc, nr, di))
672
673            if n_layers > 1 and can_via(c, r):
674                ng = g + params.via_cost
675                for nli in via_layers[li]:
676                    ns = encode(nli, c, r, -1)
677                    if ng < gscore.get(ns, math.inf):
678                        gscore[ns] = ng
679                        came[ns] = (st, (nli, c, r))
680                        heapq.heappush(heap, (ng + h(c, r), next(counter),
681                                              ns, nli, c, r, -1))
682
683        return None
684
685    # --- bounded search with widen-on-failure -------------------------------
686    # Unbounded: precompute and search the whole grid (the historical behaviour).
687    if params.search_margin is None:
688        return _search(*_precompute(0, nx - 1, 0, ny - 1))
689
690    # Bound the search to a box around the source/target nodes grown by the
691    # margin. `_precompute` fills the free mask and heuristic field only inside
692    # the box, so each bounded attempt pays for the box area, not the whole grid.
693    # On failure widen the box and retry; once it covers the grid this is the
694    # unbounded search, so a route is still found whenever one exists.
695    cs = [c for (_, c, _) in sources] + [c for (_, c, _) in targets]
696    rs = [r for (_, _, r) in sources] + [r for (_, _, r) in targets]
697    cmin0, cmax0 = min(cs), max(cs)
698    rmin0, rmax0 = min(rs), max(rs)
699    pad = max(1, int(math.ceil(params.search_margin / pitch)))
700    while True:
701        cmin, cmax = max(0, cmin0 - pad), min(nx - 1, cmax0 + pad)
702        rmin, rmax = max(0, rmin0 - pad), min(ny - 1, rmax0 + pad)
703        full = cmin == 0 and cmax == nx - 1 and rmin == 0 and rmax == ny - 1
704        path = _search(*_precompute(cmin, cmax, rmin, rmax))
705        if path is not None or full:
706            return path
707        pad *= 2

Find a min-cost node path from any source to any target.

Search internals are optimised for the Python inner loop without changing the cost model or results: search states are packed into a single integer key (((layer*ny + row)*nx + col)*9 + dir+1) instead of a tuple; a per-net boolean free mask is built once from the static occupancy so neighbour freeness is a direct array index rather than a RoutingState.is_free call; the octile heuristic is precomputed over the whole grid as a numpy field; and the via-target layer list is cached on the grid. These are transparent — the returned path is identical to the unoptimised search.

When params.search_margin is set, the search is bounded to a box around the source/target nodes (grown by that margin in mm), masking everything outside it not-free. The box widens and the search retries on failure, falling back to the full grid, so a route is still found whenever one exists; the trade-off is that a bounded route may be slightly longer than the global optimum. With search_margin=None (the default) the whole grid is searched.

Arguments:
  • state: the live routing state (occupancy the search must respect).
  • net_id: the routing net's id (nodes owned by other nets are blocked).
  • sources: candidate start nodes (layer, col, row) (a pad's accesses).
  • targets: candidate goal nodes (layer, col, row).
  • params: cost-model parameters; defaults are used when None.
Returns:

The (layer, col, row) path from a source to a target, or None if none is reachable within the expansion budget.
